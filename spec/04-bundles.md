# Resource Bundles

## Overview

Allow sellers to combine multiple existing resources into discounted "Bundles" with their own identity and pricing.

## Bundle Properties

- Title
- Description
- Cover image
- Price (discounted from individual items)
- Curriculum tags (inherited or manually specified):
  - Subject
  - Cycle
  - Competence codes

## Bundle Detail Pages

Mirror the clean layout of individual resources:

### Summary Area
- Clear summary with single main CTA (Buy/Download) in primary color

### Included Items
- Concise list of included resources
- Simple, readable stack format
- Each item shows essential info only

## Bundle Creation

### Form Design
- Stepwise, low-clutter pattern (same as single resource uploads)
- Only relevant fields shown per step

### Steps
1. Select existing resources to include
2. Set bundle title and description
3. Upload cover image
4. Set bundle price
5. Configure curriculum tags (inherit or override)
6. Review and publish
