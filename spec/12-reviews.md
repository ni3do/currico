# Reviews & Ratings

## Implementation Status

**Status: Not Implemented**

This feature is deferred to a future sprint.

---

## Overview

Allow buyers to leave star ratings and brief text reviews on purchased resources.

## Review Creation

### Eligibility
- Only buyers can leave reviews
- One review per purchase

### Review Content
- Star rating (1-5)
- Brief text comment

## Review Display

### Layout
- Simple heading for reviews section
- Minimal layout per review

### Review Item Shows
- Star rating
- Reviewer name
- Short comment
- Date

### Design Principles
- No complex threading
- No reactions or replies
- Main resource content remains visually dominant
- Reviews block separated with whitespace and subtle divider

## UI Guidelines

- Reviews don't overwhelm the page
- Clear visual hierarchy (resource info > reviews)
- Simple, scannable format
- Subtle separation from main content
