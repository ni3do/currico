# Resource Detail Pages

## Implementation Status

**Status: Complete**

All features specified below are implemented.

---

## Overview

Each resource has a clean, uncluttered detail page that highlights essential information.

## Page Structure

### Primary Information (Top Section)

- Title
- Brief description
- Price
- Primary CTA (Buy/Download) in a distinct color
- Key badges:
  - Quality status
  - Editability
  - Resource type

### Secondary Metadata Block

Displayed below primary info in a neatly spaced block:
- Subject
- Cycle
- Canton
- Lehrplan 21 competence codes

### Preview Section

- Simple gallery layout
- Watermarked or low-resolution previews
- Clearly labeled as preview content
- Full-resolution only available after purchase

### Additional Elements

- Seller info (visually secondary)
- Wishlist button (heart icon or "Save" link)
- Follow seller button
- "Report Resource" link/button near bottom of info section

### Related Resources Block

Displayed at the end of the page:
- Small list of cards for related resources/bundles
- Shows items sharing at least one common attribute
- Falls back to same-subject items if too few matches

## UI Guidelines

- Generous padding throughout
- Only a few accent colors
- Primary actions stand out without visual noise
- Report form is minimal when opened
