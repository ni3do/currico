# Seller Dashboard

## Overview

Provide a consolidated Seller Dashboard with focus on clarity over data density.

## Key Metrics (Top Section)

Displayed as large, clean stats:
- Net Earnings
- Total Downloads
- Followers

## Resources & Bundles List

### Item Display
Each row/card shows:
- Title
- Status
- Key numbers (downloads, net earnings)

### Actions
- "Edit Resource" button
- "Create Bundle" button
- Both use primary accent color
- Spaced generously to avoid crowding

## Recent Transactions View

### Columns (Limited to Essentials)
- Transaction date
- Resource/Bundle title
- Gross amount
- Platform fee
- Seller payout

### Design
- Secondary view (not primary focus)
- Clean, scannable format

## UI Guidelines

- Clarity over data density
- Large, readable key metrics
- Generous spacing between action buttons
- Primary accent color for main actions
- Secondary information available but not prominent
